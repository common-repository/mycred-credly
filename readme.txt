== Description ==

myCred-Credly lets you create, give and earn digital badges with Credly Badge Builder. With Credly API integration now you can create badges in myCred or create a badge in Credly and import it into myCred.

= myCred Credly Features =
Here are the most notable features of myCred Credly plugin.

* Admin can define if User can see only his badges that awarded or all Badges.
* Admin can create Badge image through Credly account.
* User can share his Badges on his Credly account.
* Admin can add Credly credentials in My Credly settings to enable Credly Badge builder.

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred Credly
3. Find myCred Credly and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
